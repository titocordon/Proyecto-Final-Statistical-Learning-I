Proyecto de Análisis de Datos y Modelos de Clasificación

Descripción del Proyecto:
Este proyecto se divide en tres fases principales:

Fase 1 - Análisis Exploratorio: Exploración y visualización de los datos para identificar patrones y relaciones.
Fase 2 - Ingeniería de Características: Transformación y selección de características relevantes para mejorar el rendimiento de los modelos.
Fase 3 - Modelos de Clasificación: Entrenamiento y evaluación de varios modelos de clasificación con hyper-parámetros específicos.

Requisitos
Python 3.x
Bibliotecas:
pandas
numpy
seaborn
scikit-learn
xgboost
lightgbm
datetime
matplotlib (opcional para visualizaciones)

Uso
Fase 1: Ejecuta el notebook proyecto_fase_1_AnalisisExploratorio.ipynb para realizar el análisis exploratorio de los datos.
Fase 2: Ejecuta el notebook proyecto_fase_2_IngenieriaCaracteristicas.ipynb para aplicar la ingeniería de características.
Fase 3: Ejecuta el notebook proyecto_fase_3_ModelosClasificación.ipynb para entrenar y evaluar los modelos de clasificación.


Resultados
Los resultados de cada fase se detallan en los respectivos notebooks.
El notebook de la Fase 3 incluye una tabla ordenada con los modelos evaluados, sus hiperparámetros, métricas, tiempo de entrenamiento y fecha/hora de finalización.
